API Visitway

Aca vas a poder encotrar las excursiones que mas te gusten

GET

http://localhost:3000/excursiones

Esta URL podemos obtener todo el listado de excursiones

POST

http://localhost:3000/excursiones

Aca se van a poder crear nuevas excursiones

Es necesario enviar por body el titulo , descripcion, precio y ubicacion

PUT

http://localhost:3000/excursiones/:titulo

Aca se van a poder editar las excursiones

Es necesario enviar por body el titulo , descripcion y precio

GET

http://localhost:3000/excursiones/resultados

Aca se van a poder paginar los resultados de las excursiones

GET

http://localhost:3000/excursiones/precio

Aca se van a ordenar las excursiones de menor a mayor precio

DELETE

http://localhost:3000/excursiones/:id

Aca te vas a poder eliminar la excursion de acuerdo a su ID

GET

http://localhost:3000/excursiones/:id

Aca te va a traer la excursion de acuerdo a su ID

GET

http://localhost:3000/ubicacion/:ubicacion

Aca se filtra las excursiones por ubicación

USUARIOS

GET

http://localhost:3000/usuarios

Esta URL podemos obtener todo el listado de usuarios

POST

http://localhost:3000/usuarios

Aca se van a poder crear nuevas usuarios

Es necesario enviar por body el nombre , email, password y edad

GET

http://localhost:3000/usuarios/:nombre

Esta URL podemos obtener un usuario segun su nombre

GET

http://localhost:3000/usuarios/edad/:edad

Aca se filtra las edades de los usuarios
